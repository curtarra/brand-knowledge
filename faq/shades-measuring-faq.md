# Shades Measuring FAQ

**How do I measure for inside mount shades?**
Measure width at 3 points (top/middle/bottom) — use the narrowest. Measure height at 3 points (left/middle/right) — use the longest. Frame depth must be at least 2" to fit the mounting brackets.

**How do I measure for outside mount shades?**
Width = desired coverage, extending 1–3" total beyond the window frame (1/2"–1.5" per side) for better light control. Height = desired top mounting point to the sill, adding 4–6" above the window opening so the headrail clears the frame.

**How do I measure for bay window shades?**
Mark a 2–3" projection depth in each corner angle. Measure the center panel width between the two projection marks. For side panels, measure each window's own width and add overlap only to the outermost edge for clearance.

**What are the most common shade-measuring mistakes?**
1. Measuring only once instead of three times.
2. Confusing inside-mount vs. outside-mount numbers.
3. Supplying your own deductions instead of the raw opening size.
4. Using a cloth tape or rounding to the nearest inch instead of 1/8".
5. Forgetting to account for headrail height and stack back.
Always use a steel tape and measure to the nearest 1/8".

**What is "stack back" for shades and why does it matter?**
The height a shade occupies when fully raised — Roman shades 6–18", roller shades 2–4". A large stack can block light/view; mitigate with an outside mount positioned high enough that the raised shade rests on the wall above the glass.
