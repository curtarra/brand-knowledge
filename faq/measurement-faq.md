# Measurement FAQ

**How much wider should the rod be than the window frame?**
Extend the rod 6–10" beyond the frame on each side (minimum 3" per side) so curtains stack fully off the glass when open. For sliding doors, 10–15" of extension per side is recommended.

**How high should the rod be mounted?**
4–6" above the top of the window frame for standard 8-ft ceilings, or closer to the ceiling for a taller look. Keep rod height consistent across all rods in the same room.

**How much fabric fullness do I need?**
1.5x for a flat, minimal look; 2.0x for a standard pleated look (most common); 2.0–2.5x for a rich gathered look, sheers, or bay windows.

**What tolerance should I expect since these are handmade?**
±1" on all custom dimensions.

**What about bay, sliding, or other non-standard windows?**
Measure each section/opening individually — don't average measurements across sections. Curtarra's Measurement Tool and LiveChat support can help with non-standard windows.
