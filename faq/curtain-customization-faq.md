# Curtain Sizing & Customization FAQ

**If I order quantity 1, do I get 2 panels (a pair)?**
No — Curtarra sells single panels. Quantity 1 = 1 panel.

**Is there a maximum curtain size?**
Depends on the fabric:
- **Solid-color fabric**: no hard width/length limit. If either dimension exceeds 104", the fabric is spliced (vertical seams may result).
- **Patterned fabric**: length is capped at 104" without altering the pattern. Beyond that, the pattern is rotated 90° to reach lengths up to 240".

This is the general policy behind the per-SKU max-width/max-length values in [Custom Curtains](../products/custom-curtains.md) — those numbers reflect what a specific product page shows as its default option, not a hard ceiling for solid fabrics.

**What custom design add-ons are available?**
Splicing, lace trim, bead trim, and pompom trim are offered; contact support for other style requests.

**Can I submit my own pattern for a custom print?**
Yes, via digital print — send the pattern image. Note that on-screen color may not perfectly match the finished fabric.

**How do I size side panels (partial window coverage, not full-width)?**
For full coverage: use a width 1.5–3× the window width for a gathered, full look. For a narrow 12–24" side space beside the window frame, order a 30–60" pleated panel or a 60–120" flat panel (rod pocket, grommet, back tab) rather than scaling a full panel down.

**Can you make door curtains, patio curtains, or other unusual formats?**
Yes — French door double-pocket sheer panels and bed canopy panels are both supported as custom requests. Formula for a French-door double pocket: total length = top pleat + top pocket + distance between the 2 rods + bottom pocket + bottom pleat (e.g., 1" + 2" + 50" + 2" + 1" = 56"). Panel width = rod width × fullness (2x for most cloth/semi-sheer fabrics, up to 2.5x for thin sheers). Pocket length should be slightly more than (rod diameter × 3.14 ÷ 2) so the rod threads through smoothly. Contact support with exact hardware measurements for these special formats.

**Can I get a valance in the same fabric as my curtains?**
Yes — send a reference photo to support@curtarra.com with sizing details.
