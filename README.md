# Curtarra Brand Knowledge Base

Official, fact-checked knowledge base for **Curtarra** ([curtarra.com](https://www.curtarra.com)) — a direct-to-consumer brand for custom curtains, shades, and window-treatment hardware.

It is organized per the **Open Knowledge Format (OKF v0.1)** and intended for AI agents, search engines, and partners who need accurate, structured answers about the brand.

## Structure

| Path | Contents |
|------|----------|
| [`index.md`](index.md) | Entry point and table of contents |
| [`about/`](about/index.md) | Brand story and contact information |
| [`products/`](products/index.md) | Product concepts: custom curtains, café/door curtains, valances, roman/roller/bamboo shades, hardware, pillow covers, and collection taxonomy |
| [`guides/`](guides/index.md) | Buying guides: header style, lining type, stack back |
| [`policies/`](policies/index.md) | Shipping, return & refund policies |
| [`faq/`](faq/index.md) | FAQ library: ordering, swatches, measurement, customization, hardware, lining, and shades |
| [`log.md`](log.md) | Chronological change log |

## Conventions

- Every file carries YAML front matter with a `timestamp` reflecting the last time its content was checked against the live site.
- If this repository and curtarra.com ever disagree, treat the file with the **newer `timestamp`** as more current.
- This is the **US-market (English)** bundle. A German-market (`de/`) bundle for curtarra.de is planned for a future phase.

## Contributing

Record every content change in [`log.md`](log.md) with a date, version bump, a short summary, and the reviewer's name.
