# Curtain Collection Taxonomy

Curtarra's site navigation filters the same underlying curtain catalog into four facets. These are browsing categories, not separate products with their own specs — a single fabric can appear in multiple facets at once (e.g. a velvet blackout curtain is Texture: Velvet, Function: Blackout + Scratch Resistant + Thermal).

## By Texture
Linen, Cotton, Wool, Velvet, Solid, Pattern, See Through Sheer, Semi Sheer, Quiet Luxury.

## By Function
| Collection | What it maps to |
|---|---|
| Blackout | The high end of the light-blocking range (up to 100%) — see [Custom Curtains](custom-curtains.md) |
| Room Darkening | The room-darkening-lined functional lining category — see [Lining Type](../guides/lining-type.md) |
| Semi Sheer / See Through Sheer | The low end of the light-blocking range |
| Thermal | Triple-Weave Lining plus specific wool/velvet/solid fabrics tagged thermal — see [Lining Type](../guides/lining-type.md) |
| Water Resistant | Specific outdoor fabric lines (Bensen, Clark, Edith) — an outdoor-use property, not a sitewide fabric option |
| Flame Retardant | The FR White Lining option available on select flame-retardant collections — see [Custom Curtains](custom-curtains.md) |
| Scratch Resistant | Specific velvet fabric lines (Diana, Elma) — a velvet-texture property, not sitewide |
| Room Divider | A use-case collection: heavy or ceiling-mounted panels used to split a room, drawn from the same curtain catalog |
| Double Curtains | Layering two panels on one window (e.g. sheer + blackout) using a double rod — see [Justin Adjustable Double Curtain Rods](hardware.md); not a distinct fabric line |
| Motorized Curtains | Curtains hung on the [Jason Custom Motorized Curtain Track](hardware.md) — the motorization comes from the track, not the curtain panel itself |

## By Space
Living Room, Bedroom, Dining Room, Kids Room, Outdoor, Extra Long & Wide, Odd-Shaped Window, Door Curtains, Cafe Curtains, Ceiling Curtains.

- Extra Long & Wide / Odd-Shaped Window → covered by [Custom Curtains](custom-curtains.md) sizing (no hard limit on solid fabric, splicing over 104") and the Wesley Custom Curtain Track (10"–720" span).
- Door Curtains / Cafe Curtains → dedicated files: [Door Curtains](door-curtains.md), [Café Curtains](cafe-curtains.md).
- Ceiling Curtains → ceiling mount is a confirmed hardware option (`mounting type: wall; ceiling; hanging` on applicable SKUs), not a separate fabric line.
- Kids Room → a room-based browsing collection. For documented child-safety claims, see shades (cordless, ANSI/WCMA A100.1-2018 — [Shades Overview FAQ](../faq/shades-overview-faq.md)); no equivalent certification is documented for curtains.

## By Trend
New Arrival, Best Seller, Modern, Timeless White, Neutral Tones, Side Panels, Pleated Curtains, Grommet Curtains, Valance & Swag.

- Side Panels → see [Curtain Sizing & Customization FAQ](../faq/curtain-customization-faq.md).
- Pleated Curtains / Grommet Curtains → these are header-style collections; see [Header Style](../guides/header-style.md).
- Valance & Swag → see [Valances](valances.md).
- New Arrival / Best Seller → merchandising collections that change frequently; check curtarra.com for current contents.

## Shade Collections (for comparison)
Roman Shades, Bamboo/Woven Wood Shades, Roller Shades, Motorized Shades, Blackout Shades, No-Drill Roman Shades — all already covered by [Roman Shades](roman-shades.md), [Roller Shades](roller-shades.md), and [Bamboo & Woven Wood Shades](bamboo-shades.md). Unlike curtains, shades genuinely do come in a motorized version (confirmed lift mechanism), so "Motorized Shades" is a real product variant — not the same situation as "Motorized Curtains" above.
