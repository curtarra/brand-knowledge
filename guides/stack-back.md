# Curtain Stack Back

**Stack back width** is the drape width that bunches at each edge when the curtain is fully open — minimizing it keeps more of the window/glass clear. **Stack back depth** is how far that bunched fabric projects off the wall, which matters for clearing hardware like a cornice box.

## Stack Back by Header Style

| Header Style | Stack Back Width (% of panel width) | Stack Back Depth |
|---|---|---|
| Grommet | 8–15% | 6 1/2" |
| Rod Pocket | 12–20% | 2–3" |
| Pinch Pleat / Euro Pleat | 20–25% | 3 1/2" |
| Back Tab / Tab Top | 20% | 3–4" |
| Flat Panel | 15–20% | 4–5" |
| Ripple Fold | 25–30% | 4 1/2" |

Sheer curtains have the smallest stack back; heavier fabrics stack larger. These figures are estimates — actual stack back varies with fabric weight, added lining, puddling, and the specific rod/bracket/ring hardware used.

## General Rod-Extension Rule of Thumb

To keep the stack back off the glass when curtains are open, extend the rod/track beyond the window frame:
- Standard windows: 6–10" per side (minimum 3" per side)
- Sliding doors: 10–15" per side
- Bay windows: 3–4" per section (less than a flat wall, since bay corners limit extension)
